SQL scripts used to generate analytical datasets for the project.

Scripts are organized by analytical step:

13_route_competition_summary_us_distribution.sql
    Builds route-level competition classification and route distribution.

18_route_competition_and_hhi_matrix.sql
    Computes the distribution of HHI concentration levels across route competition categories.

17_share_of_competitive_routes_with_high_hhi.sql
    Calculates the share of routes classified as competitive that still exhibit high market concentration.
